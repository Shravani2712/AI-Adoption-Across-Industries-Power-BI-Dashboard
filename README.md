📊 AI Adoption Across Industries – Power BI Dashboard
📌 Project Overview

This project analyzes AI adoption across different industries, countries, and company sizes using Power BI.
The dashboard provides insights into AI usage trends, tool popularity, and user demographics.

The goal of this project is to understand:

How AI adoption varies across industries

Which AI tools are most widely used

How adoption trends change over time

User behavior and feedback towards AI technologies

📂 Dataset Information

The dataset contains information about AI usage across multiple industries worldwide.

Key Columns
Column	Description
Industry	Industry sector using AI
Country	Country where AI is adopted
AI Tool	Tool used (ChatGPT, Bard, Claude, etc.)
Adoption Rate	Percentage of AI adoption
Daily Active Users	Number of users actively using AI tools
Company Size	Small / Medium / Large
Year	Year of adoption
Age Group	User age category
User Feedback	User satisfaction with AI tools

📈 Key Insights

Some important insights from the dashboard include:

Technology and Finance industries show the highest AI adoption rates

ChatGPT and Midjourney are among the most widely used AI tools

Large companies adopt AI faster than small companies

AI adoption has steadily increased over recent years
